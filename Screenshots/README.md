## How to Add Screenshots

1. Run the project completely
2. Take screenshots using `Windows + Shift + S` (Windows Snipping Tool)
3. Save each screenshot with the exact filename listed above
4. Place them in this `/Screenshots` folder
5. GitHub will automatically display them in the README
