# Workflow Folder

## Contents

| File | Description |
|---|---|
| `Main.xaml` | Main UiPath workflow file (open in UiPath Studio) |
| `workflow_explanation.md` | Detailed explanation of each step in the workflow |

## Note

The `.xaml` UiPath workflow file is designed to be opened in **UiPath Studio**.

Since `.xaml` files require UiPath Studio to execute, this folder contains:
- The detailed workflow explanation in Markdown format
- Step-by-step guide on recreating the workflow in UiPath Studio

## UiPath Studio Version

This workflow is compatible with **UiPath Studio 2023.x and above**.

## Quick Setup

1. Install UiPath Studio (Community Edition — free)
2. Open a new project
3. Follow the `workflow_explanation.md` to build the workflow
4. Connect it to the Dataset and MachineLearning folders
