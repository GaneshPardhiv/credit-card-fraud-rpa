# Reports Folder

This folder contains output reports generated by the fraud detection system.

## Files

| File | Description |
|---|---|
| `sample_fraud_report.csv` | Sample fraud report with 214 flagged transactions |
| `sample_output.txt` | Sample console log from a fraud detection run |
| `fraud_report.csv` | Auto-generated by `fraud_detection.py` on each run |

## Report Columns (fraud_report.csv)

| Column | Description |
|---|---|
| `Transaction_ID` | Unique transaction identifier |
| `Cardholder_Name` | Name of the card holder |
| `Card_Number` | Masked card number |
| `Timestamp` | Date and time of transaction |
| `Amount_INR` | Transaction amount in rupees |
| `Location` | Where the transaction happened |
| `Merchant_Name` | Merchant/store name |
| `Merchant_Category` | Type of merchant |
| `FraudProbability` | ML model's fraud probability (0.0 to 1.0) |
| `AlertLevel` | CRITICAL / HIGH / MEDIUM |
| `Fraud_Reason` | Rule-based reason for flagging |
| `FinalVerdict` | FRAUD or LEGITIMATE |

## Alert Levels

| Level | Fraud Probability | Action |
|---|---|---|
| CRITICAL | ≥ 0.88 | Immediate alert + block card |
| HIGH | 0.75 – 0.87 | Notify fraud team |
| MEDIUM | 0.65 – 0.74 | Log and monitor |
| LOW | < 0.65 | No action |
