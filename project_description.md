# Project Description

## Automated Credit Card Transaction Monitoring using UiPath (RPA)

**Short Description (for GitHub repo "About" section):**
> RPA + ML-powered system to automatically monitor credit card transactions, detect fraud using Scikit-learn, and generate real-time alert reports using UiPath Studio. Final Year B.Tech CSE Project – LPU 2026.

---

## Problem Statement

Every day, thousands of credit card transactions happen. Banks and financial institutions have to manually review these transactions to find fraudulent ones. This process is:

- **Slow** – humans can't check every single transaction in real time
- **Error-prone** – manual checking leads to missed fraud cases
- **Expensive** – requires a large team of analysts

**Our system solves this by automating the entire monitoring pipeline.**

---

## Proposed Solution

We combine two technologies:

1. **UiPath RPA** – handles the automation part (reading files, writing reports, sending alerts) without any human intervention
2. **Python + Scikit-learn** – handles the intelligence part (predicting whether a transaction is fraud or not)

Together, they create a complete end-to-end automated fraud detection pipeline.

---

## How This Helps Banks / Financial Companies

| Without This System | With This System |
|---|---|
| Manual review of all transactions | Automated scanning 24/7 |
| High chance of missing fraud | ML model catches patterns humans miss |
| Report generation takes hours | Report auto-generated in seconds |
| No real-time alerts | Instant alert on suspicious transaction |
| High operational cost | Significantly reduced manual effort |
